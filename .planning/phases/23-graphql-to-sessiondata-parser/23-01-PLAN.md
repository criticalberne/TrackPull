    **Main exported function `parsePortalActivity(activity: GraphQLActivity): SessionData | null`:**
    - Guard: if `!activity?.id` return null
    - Extract `reportId` via `extractActivityUuid(activity.id)`
    - Set `date` to `activity.date ?? "Unknown"`

    - Iterate `activity.strokeGroups ?? []`:
      - Skip non-object groups
      - `clubName` = `group.club || "Unknown"`
      - For each stroke in `group.strokes ?? []`:
        - Skip if `!stroke?.measurement`
        - Iterate `Object.entries(stroke.measurement)`:
          - Skip if `value === null || value === undefined` (per D-01 defensive handling, no phantom metrics)

        - Push shot only if shotMetrics has keys
      - Push club_group only if shots array non-empty, with `averages: {}`, `consistency: {}`
    - Set `metric_names` = `Array.from(allMetricNames).sort()`
    - Return session if `club_groups.length > 0`, else null
    - Wrap entire function body in try/catch, log error and return null on exception
