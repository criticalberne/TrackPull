**Find this code** (lines 386-392):
```typescript
function showStatusMessage(message: string, isError: boolean = false): void {
  const statusElement = document.getElementById("status-message");
  if (!statusElement) return;

  statusElement.textContent = message;
  statusElement.style.color = isError ? "#d32f2f" : "#388e3c";
```

**Replace with:**
```typescript
function showStatusMessage(message: string, isError: boolean = false): void {
  const statusElement = document.getElementById("status-message");
  if (!statusElement) return;

  statusElement.textContent = message;
  statusElement.classList.remove("status-error", "status-success");
  statusElement.classList.add(isError ? "status-error" : "status-success");
}
```

**What changed:**
- Removed: `statusElement.style.color = isError ? "#d32f2f" : "#388e3c";`
- Added: `statusElement.classList.remove("status-error", "status-success");` to clear previous class
- Added: `statusElement.classList.add(isError ? "status-error" : "status-success");` to set new class

The CSS classes `.status-error` and `.status-success` are defined in popup.html (added by Plan 09-01) and use `var(--color-status-error)` and `var(--color-status-ok)` tokens that respond to the dark mode @media query.

- The function signature

- Any other function in popup.ts
