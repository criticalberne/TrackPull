# Plan 05-01: TSV Writer Module

## Goal

Create a `writeTsv()` function that converts `SessionData` into a tab-separated string with column headers and unit labels. Shots only -- no averages or consistency rows. Mirrors the structure of `csv_writer.ts` with tab delimiter and TSV-specific field escaping.

## Tasks

<task id="A1" title="Create src/shared/tsv_writer.ts with writeTsv function">
Create the TSV writer module at `src/shared/tsv_writer.ts`.

Implementation details:
- Import `SessionData` from `../models/types`
- Import `getApiSourceUnitSystem`, `getMetricUnitLabel`, `normalizeMetricValue`, `DEFAULT_UNIT_CHOICE`, `type UnitChoice` from `./unit_normalization`
- Import `METRIC_DISPLAY_NAMES` from `./constants`
- Define `METRIC_COLUMN_ORDER` as a module-level constant (same values as in `csv_writer.ts` -- do NOT import from csv_writer)
- Implement `escapeTsvField(value: string): string` -- replace `\t` with space, replace `\n` and `\r` with space
- Implement `getDisplayName(metric: string): string` -- same pattern as csv_writer
- Implement `getColumnName(metric: string, unitChoice: UnitChoice): string` -- same pattern as csv_writer, returns display name with unit label in parentheses
- Implement `orderMetricsByPriority(allMetrics: string[], priorityOrder: string[]): string[]` -- same pattern as csv_writer
- Export `writeTsv(session: SessionData, unitChoice?: UnitChoice): string`

`writeTsv` logic:
1. Get ordered metrics using `orderMetricsByPriority(session.metric_names, METRIC_COLUMN_ORDER)`
2. Build header row: `["Date", "Club", "Shot #"]` then metric columns with unit labels via `getColumnName()`
3. No Tag column, no Type column
4. Get source unit system via `getApiSourceUnitSystem(session.metadata_params)`
5. Loop through `session.club_groups` -> `club.shots` only (skip averages, skip consistency)
6. For each shot, build a row: `[session.date, club.club_name, String(shot.shot_number + 1)]` then each metric value normalized via `normalizeMetricValue()`
7. Apply `escapeTsvField()` to every field value before joining
8. Join fields with `\t`, join rows with `\n`
9. Return header row + data rows joined with `\n`

All functions use named exports (no default exports).
</task>

<task id="A2" title="Create tests/test_tsv_writer.ts with unit tests" depends_on="A1">
Create the test file at `tests/test_tsv_writer.ts` using vitest `describe/it/expect`.

Import `writeTsv` from `../src/shared/tsv_writer`, `SessionData` from `../src/models/types`, `UnitChoice` from `../src/shared/unit_normalization`.

Create a `makeSession` helper (same pattern as `test_csv_writer_units.ts`) that accepts `metrics`, `metricNames`, and optional `metadataParams` (default `{ nd_001: "789012" }`). Also create a `makeMultiClubSession` helper for multi-club tests.

Test groups:

**describe("TSV Writer: tab delimiter")**
- `it("uses tabs as column delimiter")` -- split header on `\t`, verify column count > 1
- `it("does not contain commas as delimiters")` -- verify the header row does not use comma-delimited columns

**describe("TSV Writer: header row")**
- `it("starts with Date, Club, Shot # columns")` -- split header on `\t`, verify first 3 columns
- `it("includes unit labels in metric column headers")` -- verify `Club Speed (mph)` appears for imperial
- `it("does not include Type column")` -- verify header does not contain a standalone "Type" column
- `it("does not include Tag column")` -- verify header does not contain "Tag"

**describe("TSV Writer: shots only")**
- `it("does not include average rows")` -- create session with averages, verify no row contains "Average"
- `it("does not include consistency rows")` -- create session with consistency data, verify no row contains "Consistency"
- `it("outputs one data row per shot")` -- 3 shots in session -> 4 lines total (1 header + 3 data)

**describe("TSV Writer: edge cases")**
- `it("replaces tab in field value with space")` -- club name containing `\t`, verify no extra columns
- `it("replaces newline in field value with space")` -- metric value containing `\n`, verify correct row count
- `it("does not escape commas in field values")` -- club name with comma, verify it survives unquoted
- `it("handles empty session with no clubs")` -- empty `club_groups`, verify output is header row only
- `it("handles shot with missing metric values")` -- shot with empty metrics object, verify empty string values

**describe("TSV Writer: unit conversion")**
- `it("converts speed from m/s to mph for imperial")` -- 44.704 m/s -> ~100 mph
- `it("keeps speed in m/s for metric")` -- 44.704 m/s -> ~44.7
- `it("converts distance from meters to yards for imperial")` -- 182.88m -> 200 yds

**describe("TSV Writer: multi-club sessions")**
- `it("includes shots from all clubs in order")` -- 2 clubs with 2 shots each -> 5 lines total (1 header + 4 data)
</task>
